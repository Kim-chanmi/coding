함수

함수는 자바스크립트에서 기본적인 구성 중의 하나입니다. 작업을 수행하거나 값을 계산하는 문장 집합 같은 자바스크립트 절차입니다. 함수를 사용하려면 함수를 호출하고자 하는 범위 내에서 함수를 정의해야 합니다.

01. 선언적 함수

함수를 정의 후, 호출하는 일반적인 함수입니다. 함수의 앞 또는 뒤에서 호출 가능합니다. 변수나 함수를 선언하지 않고 호출하였을 때는 에러가 발생하지만, var로 선언한 변수와 선언적 함수는 변수, 함수 선언 이전에 호출해도 에러를 발생시키지 않습니다. 이러한 자바스크립트 성질을 호이스팅(Hoisting이라고 합니다.

다음은 선언적 함수의 형식입니다.

function func(){
    document.write("함수가 실행되었습니다.1");
}
func();

02. 익명 함수

익명 함수는 변수에 함수 데이터를 저장하여 변수를 마치 함수처럼 사용 할 수 있도록 만들어 줍니다. 익명 함수는 변수 선언 이후에 호출해야 합니다.

다음은 익명 함수의 형식입니다.

const func = function(){
    document.write("함수가 실행되었습니다.2");
}
func();

03. 매개변수 함수

함수에 변수값이 전달되어 처리해야 하는 경우가 있는데, 이렇게 함수에 전달되는 매개변수라고 합니다. 이때, 변수 선언은 생략합니다. 매개변수가 있는 함수의 경우 개수가 정해져 있습니다. 하지만 경우에 따라서는 개수를 정할 수 없는 함수를 만들어야 하는 경우도 발생합니다. 이런 경우 함수에 전달되는 매개변수 값을 저장해주는 객체(arguments)를 사용하여 함수를 만들 수 있습니다. 여기서 객체는 배열의 형태로 저장됩니다.

다음은 매개변수 함수의 형식입니다.

function func(str){     //변수선언 생략함
    document.write(str);
}
func("함수가 실행되었습니다.3");

04. return 함수

함수를 통해 처리된 결과를 반환시켜주는 명령어입니다. 함수 안에서 return을 만나게 되면 해당 함수를 호출한 곳으로 결과 데이터를 반환해 주고 함수는 종료가 됩니다.

다음은 return 함수의 형식입니다.

function func(){
    const str = "함수가 실행되었습니다.4";
    return str;
}
document.write(func());

05. 화살표 함수

ECMAScript6(ES6)에 추가된 내용으로 '=>'을 이용하여 함수를 간결하게 표현할 때 사용합니다. function 키워드를 생략하하고 '='과 '>'을 합쳐 코딩합니다. 또한, 약식으로도 표현할 수 있습니다.

다음은 선언적 함수를 화살표 함수로 실행한 것입니다.

//화살표함수
func = () => {
    document.write("함수가 실행되었습니다.5")
}
func();
