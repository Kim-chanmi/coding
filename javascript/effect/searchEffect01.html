indexOf()를 이용하여 검색하기

HTML 태그 검색하기

  • a : a 태그는 다른 페이지 이동을 설정합니다.
  • abbr : abbr 태그는 줄임말이나 머리글자를 표현하는데 사용합니다.
  • acronym : acronym 태그는 줄임말이나 머리글자를 표현하는데 사용합니다.
  • address : address 태그는 웹 페이지의 연락처 정보를 설정합니다.
  • applet : applet 태그는 애프렛 요소를 설정합니다. 현재는 사용하지 않습니다.
  • area : area 태그는 이미지 맵의 영역을 설정합니다.
  • article : article 태그는 문서, 페이지, 애플리케이션, 또는 사이트 안에서 독립적으로 구분해 배포하거나 재사용할 수 있는 구획을 나타냅니다.
  • aside : aside 태그는 문서의 주요 내용과 간접적으로만 연관된 부분을 나타냅니다. 주로 사이드바 혹은 콜아웃 박스로 표현합니다.
  • audio : audio 태그는 오디오 파일을 설정합니다.
  • b : b 태그는 글자를 굵게 표시하는 태그로, bold의 약자입니다.
  • base : base 태그는 모든 문서의 기본이 되는 URL을 설정합니다.
  • bdi : bdi 태그는 주위의 텍스트들과는 다른 방향으로 써지는 텍스트 영역을 정의할 때 사용합니다.
  • bdo : bdo 태그는 현재 텍스트의 쓰기 방향을 덮어쓰고 다른 방향으로 렌더링 할 때 사용합니다.
  • bgsound : bgsound 태그는 페이지가 사용되는 동안 배경에서 재생할 사운드 파일을 설정합니다.
  • big : big 태그는 텍스트의 크기를 크게 설정합니다.
  • blink : blink 태그는 텍스트를 깜박이게 합니다.
  • blockquote : blockquote 태그는 안쪽의 텍스트가 긴 인용문임을 나타냅니다. 주로 들여쓰기를 한 것으로 그려집니다.
  • body : body 태그는 HTML 문서의 내용을 나타냅니다. 한 문서에 하나의 <body> 요소만 존재할 수 있습니다.
  • br : br 태그는 텍스트 안에 줄바꿈을 생성합니다.
  • button : button 태그는 클릭 가능한 버튼을 나타냅니다. 버튼은 양식 내부는 물론 간단한 표준 버튼 기능이 필요한 곳이라면 문서 어디에나 배치할 수 있습니다.
